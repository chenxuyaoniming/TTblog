### flex布局（弹性自适应）

#### display: flex

#### flex-direction
设置主轴方向

#### flex-wrap
设置元素是否可以换行

#### justify-content
设置主轴方向的元素位置排布

#### align-items
设置交叉轴上元素的排布（单行）

#### align-content
设置多轴元素的排布

#### flex: grow(放大比例) shrink(缩小比例) basist(基础大小)
设置元素相对于剩余控件的缩放比例 flex: auto == 0 1 auto flex: 1 == 1 1 auto
